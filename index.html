My Favorite Tools For Software Testing

Java is my favorite programming language. It uses object oritented programming concepts. Over 3 billion devices are running on Java. I use the Java framworks JUnit and TestNG in conjuction with Selenium WebDriver to develop my test scripts.

Selenium WebDriver is my top choice when it comes to automation testing. This tool is really simple and easy to use, when used with programming best practices it makes testing and development simple.

Jira is a powerful task management tool. Jira allows us to plan out tasks for a sprint using the Jira board. Jira also comes with analytical tools to generate charts showing the progression of work done and work remaining.

Cucumber is an awesome and really easy to use tool. Using keywords such as GIVEN WHEN AND THEN we can write our test steps in a feature file provided by Cucumber.
